# Security Policy

## Supported Versions

Only the current actively maintained branch in this repository should be considered supported.

| Version | Supported |
| ------- | --------- |
| current `main` | yes |
| older snapshots | no |

## Reporting a Vulnerability

Please do not disclose exploitable security issues in a public issue first.

Recommended order:

1. Contact the current maintainer through the current GitHub repository owner channel.
2. If you cannot establish a private contact, open a minimal public issue without exploit details and request a private follow-up.

Repository:

- `https://github.com/Acooldog/fuckschoolrun`

When reporting, include:

- affected app version
- Android version / ROM
- reproduction steps
- impact
- suggested fix, if available

## Notes

This repository is independently maintained now. However, it still contains GPL-licensed code evolved from upstream open-source work, so security fixes should preserve the required license notices.
